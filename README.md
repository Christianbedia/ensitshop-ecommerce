# ensitshop-ecommerce

Application e-commerce développée avec HTML, CSS et JavaScript.

## Fonctionnalités

- Catalogue de produits avec images
- Filtrage par catégorie (Électronique, Vêtements, Livres, Maison)
- Recherche de produits
- Tri par prix et nom
- Panier d'achat avec localStorage
- Gestion des quantités
- Design responsive pour mobile
- Prix en Francs CFA (FCFA)

## Installation

1. Clonez le dépôt
2. Ouvrez `index.html` dans votre navigateur

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome pour les icônes
